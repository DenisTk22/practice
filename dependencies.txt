- react приложения
    - создание в текущуей папке: npx create-react-app .
    - создание в папке my-app: npx create-react-app my-app
        - переход в папку my-app: cd my-app
    - запуск приложения: npm start
- mobx
    - установка с поддержкой только функциональных компонентов (-lite): npm i mobx mobx-react-lite
    - установка с поддержкой функциональных и классовых компонентов: npm i mobx mobx-react
