- react приложения
    - создание в текущуей папке: npx create-react-app .
    - создание в папке my-app: npx create-react-app my-app
        - переход в папку my-app: cd my-app
    - запуск приложения: npm start
- mobx
    - установка с использованием функциональных компонентов (-lite): npm i mobx mobx-react-lite
    - установка с использованием классовых компонентов: npm i mobx mobx-react
